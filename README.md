# Maya Pipeline Tools
A collection of Python utility scripts for Maya, built to support
layout and technical art pipelines in VFX and games production.

## Scripts

### geo_renamer.py
Adds a GEO_ prefix to all selected objects in the scene.
Enforces naming conventions before handoff to Animation and Lighting.

### scene_stats.py
Prints a quick health summary of the current Maya scene —
mesh count, polygon count, cameras, referenced files, and unknown nodes.

### freeze_and_clean.py
Freezes transforms and deletes construction history on selected objects.
One-click prep before handing geometry off to rigging or animation.

## Usage
Open Maya's Script Editor (Python tab), paste the script, and run.
Each script is self-contained and safe to run at any production stage.
